# Sistema de Estoque em Java

Projeto de estudo em Java para praticar **Programação Orientada a Objetos (POO)**,
arrays e lógica de controle.

## Funcionalidades
- Cadastro de produtos
- Cálculo do valor total do estoque
- Exibição de todos os produtos cadastrados

## Estrutura do projeto
- `model/Produto.java` → Representa os produtos
- `service/EstoqueService.java` → Lógica de estoque
- `Main.java` → Interação com o usuário (console)

## Tecnologias
- Java
- Scanner
- POO
- Arrays
